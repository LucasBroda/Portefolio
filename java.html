Projets Java

Je vais également vous présenter deux projets réalisés en Java lors de mon BUT Informatique.
Tout d'abord, j'ai réalisé lors de mes SAE (projets de groupe notés), une application de gestion de vente de produits.
Cette application permet en effet divers actions :
- Ajouter un produit au stock
- Supprimer un produit du stock actuel
- Ajouter ou supprimer un nouveau client du répertoire des clients

Lors d'une autre SAE, toujours en groupe, j'ai également pû réaliser une petite application permettant de générer différentes fractales (Une figure fractale est un objet mathématique qui présente une structure similaire à toutes les échelles.)
En effet, grâce à une classe spécialement créée pour cela, on peut générer des images représentant différentes suites.
Mon groupe et moi, avons réussi à créer des représentations de :
- La suite de Julia
- La suite de Mandelbrot
- Les deux suites avec des expressions générales

Ainsi, lorsqu'on lance cette classe, on pourra retrouver à la racine du projet, les divers images générées.
On peut, si on le désire, changer la couleur de l'image générée, sa taille, on peut également remplacer la formule (pour les suites généralisées), pour obtenir d'autres images.
