Projets PHP

Pour ce qui est des projets, lors d'une SAE, j'ai pû créer une application (sans front) en PHP, comportant un système d'authentification (on peut créer un compte, se connector se déconnecter) utilisant Fortify.
Une fois connecté, l'utilisateur peut avoir accès à une liste d'oeuvres avec plusieurs détails (auteur, date de création, description).
L'utilisateur connecté a également la possibilité d'ajouter une nouvelle oeuvre si il le désire à la base de données du site.
Enfin, le site donne également à l'utilisateur de consulter son profil et de modifier, si il le souhaite, son avatar ou les détails qu'il souhaite.
